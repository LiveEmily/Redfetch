## Dependencies
 - `wmctrl`
 - `a C compiler` (such as clang or gcc)
 - `make`
## Installation
Build:
```
git clone https://github.com/RedsonBr140/redfetch.git
cd redfetch
make
```
Install:
```
sudo make install
```
